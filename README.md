# Music_DS
## Работа с музыкальными нотами гитар
### Часть: Загрузка, предобработка и подготовка данных:
1.	Загрузка аудиофайлов. Загружены все файлы Калибровочного набора
2.	Получить описательные статистики для каждого микрофона в Калибровочном наборе
3.	Получить спектрограммы для каждого микрофона из Калибровочного набора.
4.	Визуализировать полученные результаты для каждого микрофона в Калибровочном наборе.
5.	Указать особенности каждого микрофона в Калибровочном наборе, отличающего его от других (отличительные характеристики)
6.	Сохранить характеристики всех типов микрофонов в файле формата JSON
7.	Получить описательные статистики для всех аудиофайлов в Калибровочном наборе.
8.	Получить спектрограммы для всех аудиофайлов в Калибровочном наборе.
9.	Визуализировать описательные статистики и спектрограммы для всех аудиофайлов в Калибровочном наборе

### Часть: Разведочный анализ данных:
1.	Выполнить загрузку всех аудиофайлов основного набора.
2.	Выделить сигналы (временные нарезки по длительности звучания ноты) по ладам
3.	для каждого инструмента.
4.	Выделить сигналы струн (временные нарезки) для каждого лада для каждого
5.	инструмента.
6.	Получить длительности звучания нот и выполнить их визуализацию.
7.	Сформировать итоговый набор предобработанных данных, который содержит
8.	разметку для всех инструментов, по всем ладам, струнам, расстояниям до источника
9.	звука.
10.	Сохранить итоговый набор предобработанных данных в JSON-формате
11.	Получены описательные статистики, гистограммы для настроенных и расстроенных
12.	инструментов.
13.	Визуализированы описательные статистики, гистограммы для настроенных и
14.	расстроенных инструментов.
15.	Выделены характеристики сигнала, позволяющие отличать звучание настроенного
### Часть: Построение, обучение и оптимизация модели (классификация гитар по звуку)
